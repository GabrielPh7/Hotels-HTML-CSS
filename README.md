## :checkered_flag: Starting

```bash
# Clone this project
$ git clone https://github.com/GabrielPh7/projeto_hotels

# Access
$ cd projeto_hotels

# Install dependencies
$ yarn

# Run the project
$ yarn start

# The server will initialize in the <http://localhost:3000>
```
